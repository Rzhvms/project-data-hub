# PasswordEncryptionService

`PasswordEncryptionService` — сервис для безопасного хеширования и проверки паролей на базе **Argon2id**.

Он реализует интерфейс `IPasswordEncryptionService` и предназначен для использования в подсистеме аутентификации и хранения учетных данных.

---

## Назначение

Сервис решает две задачи:

1. Создает криптографически стойкий хеш пароля.
2. Проверяет введенный пароль на соответствие ранее сохраненному хешу.

Для этого используется алгоритм **Argon2id**, который считается одним из наиболее безопасных вариантов для хранения паролей.

---

## Используемые зависимости

Сервис опирается на следующие пространства имен и библиотеки:

- `System.Security.Cryptography`
- `System.Text`
- `Konscious.Security.Cryptography`

---

## Основные параметры

В сервисе заданы фиксированные параметры хеширования:

```csharp
private const int SaltSize = 16;
private const int HashSize = 32;

private const int Iterations = 2;
private const int MemorySize = 19456;
private const int Parallelism = 1;
```
### Описание параметров
SaltSize — размер соли в байтах.  
HashSize — размер итогового хеша в байтах.  
Iterations — число проходов алгоритма.  
MemorySize — объем памяти, используемый Argon2id.  
Parallelism — степень параллелизма.  

---

## Формат сохранения хеша

Метод HashPassword возвращает строку в формате:
```csharp
argon2id:{iterations}:{memorySize}:{parallelism}:{salt}:{hash}
```
Где:

argon2id — имя алгоритма;  
iterations — количество итераций;  
memorySize — объем памяти;  
parallelism — степень параллелизма;  
salt — соль в формате Base64;  
hash — итоговый хеш в формате Base64.  

---

## Метод HashPassword
```csharp
public string HashPassword(string password)
```
### Назначение
Создает хеш пароля с использованием случайно сгенерированной соли.

### Логика работы
1. Проверяет входной параметр `password` на `null`.
2. Генерирует случайную соль с помощью RandomNumberGenerator.
3. Создает экземпляр Argon2id.
4. Устанавливает соль, число итераций, объем памяти и параллелизм.
5. Вычисляет хеш фиксированной длины.
6. Возвращает сериализованную строку с параметрами и результатом хеширования.

### Особенности

- Соль создается заново для каждого пароля.
- Результат можно безопасно хранить в базе данных.
- Строка содержит все параметры, необходимые для последующей проверки.

--- 

## Метод VerifyPassword
```csharp
public bool VerifyPassword(string password, string storedHash)
```
### Назначение

Проверяет, соответствует ли введенный пароль сохраненному хешу.

### Логика работы
1. Проверяет входные параметры `password` и `storedHash` на `null`.
2. Разбивает сохраненный хеш по символу :.
3. Проверяет корректность формата строки.
4. Извлекает параметры алгоритма.
5. Декодирует соль и ожидаемый хеш из Base64.
6. Создает новый экземпляр Argon2id с теми же параметрами.
7. Вычисляет хеш от переданного пароля.
8. Сравнивает полученный хеш с ожидаемым через CryptographicOperations.FixedTimeEquals.

## Особенности безопасности

Для сравнения хешей используется:
```csharp
CryptographicOperations.FixedTimeEquals(actualHash, expectedHash)
```
Это защищает от timing attacks, так как сравнение выполняется за фиксированное время.

## Обработка ошибок

Методы работают безопасно и не выбрасывают исключения при невалидных входных данных в процессе проверки.

### `HashPassword`
- Выбрасывает исключение, если password == null.
### `VerifyPassword`
Возвращает false, если:
- строка хеша имеет неправильный формат;
- алгоритм не argon2id;
- не удается распарсить параметры;
- не удается декодировать Base64;
- хеши не совпадают.

## Безопасность

Реализация учитывает базовые требования к хранению паролей:

- используется стойкий алгоритм Argon2id;
- для каждого пароля создается уникальная соль;
- параметры хеширования сохраняются вместе с хешем;
- сравнение выполняется без утечки по времени;
- пароль не хранится в открытом виде.
